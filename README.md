# Agentic AIOps Labs

This repository contains the 2 labs for Agentic AIOps in AIOps Module:

## Lab 1: AI Agent Connection with MCPs
Demonstrates how AI agents can connect and interact with Model Context Protocol (MCP) servers to access external systems and data sources. This lab focuses on establishing communication channels between agents and various backend services through standardized protocols.

## Lab 2: Agent-to-Agent Communication
Showcases the implementation of Agent-to-Agent (A2A) intercommunication protocols for distributed AIOps scenarios. This lab explores how multiple specialized agents can coordinate and collaborate to automate ITIL service management processes, including monitoring, event management, and ticketing workflows.
